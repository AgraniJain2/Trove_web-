# Trove Web — AI-Powered Content Vault
A responsive web app for organizing text, links, images & files with ML auto-tagging and semantic search.

## Features
- 📝 Save text, links, images, and files
- 🤖 ML auto-tagging (TF-IDF keyword extraction)
- 🔍 Semantic search (cosine similarity + synonym expansion)
- 🌙 Dark/Light theme
- 💾 localStorage persistence
